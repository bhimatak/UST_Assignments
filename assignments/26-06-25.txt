1. Given an array  consisting of  positive and negative integers then without changing the  relative  order of  elements.  
  
   For example :
   Input :  
    8
	9 -3 5 -2 -8 -6 1 3
   Output : 
   -3 -2 -8 -6 9 5 1 3   


2. Given  an binary  array,  Sort it in linear time and constant space. Output  Should Print contain all zeroes followed  by all ones.

For  example:
Input : 
 8
1 0 0 0 1 0 1 1
Output:
0 0 0 0 1 1 1 1


3.    Given an  array of  integers, move all zeros present in the array  to   the end.  The solution should maintain the relative order of  items  in the  array.

For example,

Input :  
9
6 0 8 2 3 0 4 0 1
Output:
6 8 2 3 4 1 0 0 0
